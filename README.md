# vivid-design-tokens

Design tokens are the prime dependency of our design system

In order to release a new version of the package do:
- update the version in `package.json`, commit and push
- go to the repository's [Actions page](https://github.com/Vonage/vivid-design-tokens-properties/actions) in GitHub UI (can use the link here)
- select 'Publish Design Token Properties'
- go to the last build
- hit re-run the build
